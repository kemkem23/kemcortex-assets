# kemcortex-assets

Public asset host for images and files produced inside the **private** kemcortex repos
(e.g. cortex-orchestrator) that need to be shown to the user via a chat client.

Why this exists: the user's chat client cannot fetch from `localhost`,
`http://kemcortex:<port>`, or filesystem paths on the kemcortex server.
GitHub raw URLs from a public repo are the only reliable inline-image path.

## Layout

- `codex/` — images generated by `codex exec` (image_generation tool)
- (more subdirs as needed)

## Linking

```
https://raw.githubusercontent.com/kemkem23/kemcortex-assets/main/<path>
```

This repo is intentionally public. Do **not** commit secrets, credentials, or
private user content here.
